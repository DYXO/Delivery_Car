# Delivery Car

## Get start
### Step 1 install pigpio
go to this page and follow it
https://abyz.me.uk/rpi/pigpio/download.html
### Step 2 clone this Project
```
git clone 
```
### Step 3 cd to the directory
```
cd build
```
### Step 4 build
```
cmake ../
```
```
make
```
### Step 5 run the project
```
sudo ./DeliveryRobot
```
